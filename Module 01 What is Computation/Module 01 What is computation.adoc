## Numeric 예제
* 숫자 x의 제곱근은 y. y * y = x와 같음
* 숫자 x의 제곱근을 구하는 방법
** g라는 추측으로 시작
** 만약 g * g가 x의 제곱근 조건을 만족하면, 계산을 멈추고 g가 답이라고 말함
** 아니면, g와 x를 g의 평균으로 새로운 추측
** 새 추측을 사용하여 제곱근 조건을 만족할 때 까지 반복

[cols=4, options=header]
|===
|g|g * g|x / g|(g + x / g) / 2
|3|9|16 / 3|4.17
|4.17|17.36|3.387|4.0035
|4.0035|16.0277|3.997|4.000002
|===
