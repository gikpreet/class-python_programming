## 컴퓨터는 어떤 일을 하는가?
* 기본적으로:
    * 계산을 수행함: 1초에 수십 억 번의 계산을 수행
    * 결과를 저장: 수백 Tera byte의 저장 공간을 가짐
* 어떤 종류의 계산을 수행하는가?
    * 프로그래밍 언어에 미리 포함된(built-in) 계산
    * 프로그래머가 정의한 계산
* 컴퓨터는 이야기 하지 않은 것을 알지 못함
