== 클래스 구현 vs. 클래스 사용

* 두 가지 다른 관점에서 코드 작성

* 클래스 구현 관점
** 클래스를 사용하여 새로운 객체 타입 **구현**
*** 클래스 **정의**
*** **데이터 속성** 정의 (그 객체는 무엇인가)
*** **메소드** 정의 (객체를 사용하는 방법)

* 클래스 사용 관점
** 코드에서 새 객체 타입 **사용**
*** 객체 타입의 인스턴스 생성
*** 생성한 인스턴스로 연산(operations) 수행
